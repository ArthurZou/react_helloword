# About & Stack

AI engineer building at the intersection of generative models and developer experience.
I care about fast iteration loops, minimal abstractions, and shipping things that work.

## Background

Started in backend systems, drifted toward ML infrastructure, ended up fully in the world of
diffusion models and LLM tooling. Currently focused on making AI-assisted workflows feel
native to the developer experience.

## Dev Workflow

```bash
# spin up local stack
docker compose up -d

# run model server with hot reload
uvicorn app.main:app --reload --port 8000

# run evals against golden set
python -m pytest tests/evals/ -v --tb=short
```
